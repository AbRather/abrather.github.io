# Reference Letters Directory

## Instructions

Place your reference letter PDF files in this directory with the following naming convention:

### Naming Convention:
- Use the job title/position as the filename
- Replace spaces with underscores or hyphens
- Use lowercase for consistency
- Add `.pdf` extension

### Examples:

**For: "Studentische Hilfskraft"**
- Filename: `studentische_hilfskraft.pdf`

**For: "Werkstudent - Flight Demonstrator Assembly & Component Testing"**
- Filename: `werkstudent_flight_demonstrator.pdf`

**For: "Propulsion Test Data Intern"**
- Filename: `propulsion_test_data_intern.pdf`

**For: "Propulsion & Design Intern"**
- Filename: `propulsion_design_intern.pdf`

**For: "Werkstudent - Advanced Computational Laboratory"**
- Filename: `werkstudent_advanced_computational.pdf`

### Current Reference Files:

Add your PDF files here. The website will automatically detect if a reference is available.
If a reference PDF is not present, the badge will not appear for that position.

### File Organization:

```
/static/pdf/references/
  ├── README.md (this file)
  ├── studentische_hilfskraft.pdf (Faserinstitut Bremen)
  ├── werkstudent_flight_demonstrator.pdf (Polaris Spaceplanes)
  ├── propulsion_test_data_intern.pdf (Rocket Factory Augsburg)
  ├── propulsion_design_intern.pdf (SpanTrik)
  └── werkstudent_advanced_computational.pdf (SRM Institute)
```

### Notes:
- PDFs should be optimized for web (compressed)
- Recommended file size: Under 2MB per file
- Ensure PDFs are properly formatted and professional
- Test that PDFs open correctly in browsers
